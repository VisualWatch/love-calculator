## 🔐 Firebase Setup (Optional)

If you want to store data:

1. Go to Firebase Console
2. Create a project
3. Enable Firestore Database
4. Replace config in `script`:

```js
const firebaseConfig = {
  apiKey: "YOUR_KEY",
  authDomain: "YOUR_DOMAIN",
  projectId: "YOUR_PROJECT_ID"
};
```
